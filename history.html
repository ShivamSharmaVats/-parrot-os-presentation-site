Parrot OS

Computer System Overview / History

Next: Requirements

Who Created It?

  • Parrot Security team (founder/team lead: Lorenzo Faletra).
  • First release: 2013.
  • Debian-based and focused on security, privacy, and development.

How Is It Distinct?

  • Security and forensic toolsets in Security Edition.
  • Privacy-oriented defaults and hardening options.
  • Multiple editions: Home, Security, Raspberry Pi, Docker, WSL.

Picture/Video of It Running

Legacy

Why Was/Is It Important?

  • Helped students enter cybersecurity through ready-to-use labs.
  • Balanced pro security tooling with a general desktop option.
  • Strong community visibility in CTF and security training.

Is It Still Used?

  • Yes, still actively maintained.
  • Used in VMs, labs, and cyber education environments.
  • Project reports very high global download counts.
