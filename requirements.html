System Requirements & Compatibility

Next: Security

What Does It Need To Run?

Minimum

  • 64-bit CPU (x86_64/amd64)
  • 4 GB RAM
  • 40+ GB storage
  • USB drive for live boot/install

Recommended

  • Quad-core 64-bit processor
  • 8-16 GB RAM
  • 80+ GB SSD
  • Virtualization support (VT-x/AMD-V)

Availability / Target Devices

Device Types

  • Laptops and desktops
  • Virtual machines
  • WSL on Windows
  • Raspberry Pi builds

CPU Compatibility

  • Mainstream: x86_64
  • ARM supported in specific builds
  • Power/RISC-V not common official targets

File System Support

  • ext4 (common default)
  • NTFS, FAT32, exFAT interoperability
  • Other Linux file systems through kernel support

GPU Compatibility

  • NVIDIA RTX and legacy NVIDIA with Linux drivers
  • AMD Radeon modern stack supported
  • Enterprise GPU virtualization is possible in special setups

Connectivity

  • USB, Ethernet, Wi-Fi, Bluetooth
  • WWAN possible with compatible hardware
  • VGA/HDMI/DisplayPort depends on device outputs
